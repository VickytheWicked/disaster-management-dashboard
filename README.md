## Overview
A full-stack disaster response coordination platform designed to manage volunteers, warehouses, and resource distribution.

The system models real-world disaster workflows with role-based access and inventory tracking.

---

## Tech Stack
- React (Frontend)
- Node.js (Backend)
- SQLite (Database)
- RESTful API

---

## Architecture
Frontend (React) ‚Üî Backend (Node.js API) ‚Üî SQLite Database

---

## Features
- Role-based access control (Admin, Volunteer, Warehouse Owner)
- Resource inventory tracking
- Warehouse management system
- Announcement & communication module
- Structured relational database schema

---

## Setup
1. Install dependencies
2. Configure database
3. Start backend server
4. Run frontend client

---

## Future Improvements
- JWT-based authentication
- WebSocket-based real-time chat
- Cloud deployment
